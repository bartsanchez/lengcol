# Lenguaje coloquial project

# Requirements

virtualenvwrapper
docker
docker-compose

# Installation

```sh
$ mkvirtualenv lengcol

(lengcol) $ make build
(lengcol) $ make run
```

# Testing

Simply, run:

```sh
$ make test
```
