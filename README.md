# Lenguaje coloquial project

# Requirements

virtualenvwrapper
docker
docker-compose

# Installation

```sh
$ cp example_env .env

$ mkvirtualenv lengcol

(lengcol) $ make build
(lengcol) $ make run
```

# Testing

Simply, run:

```sh
$ make test
```
