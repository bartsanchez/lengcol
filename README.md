# Lenguaje coloquial project

# Requirements

virtualenvwrapper

# Installation

```sh
$ mkvirtualenv lengcol

(lengcol) $ pip install -r requirements/base.txt
```

# Testing

Simply, run:

```sh
$ make test
```
